# Pay2Slay Faucet

## Configs
- configs/payout.yaml — payout_amount_ban_per_kill, scheduler_minutes, caps
- configs/integrations.yaml — Discord/Yunite/Fortnite, Banano node, dry-run, wallet floor
- configs/product.yaml — Whitelabel branding & flags

## Privacy & Eligibility
